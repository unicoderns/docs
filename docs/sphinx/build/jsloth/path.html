JSloth Path¶

Helper that loads the right view for each app.

Public Functions¶

Get¶

Get the path for the Angular compiled file:

get(app: string, file: string)
Arguments:
  • folder (string) – App folder requesting the view
  • name (string) – App requesting the view
  • file (string) – Name of the file to load
Returns:

string

Example:¶

1
 let path = this.jsloth.path.get(this.config.folder, this.config.name, file);
